Metodologia de trabajo

La implementación adecuada es esencial para lograr un sistema confiable y que cumpla con las necesidades de la organización. Este proceso abarca todos los métodos y actividades realizadas para que la implantación de un sistema sea exitosa. La implementación puede lograrse mediante los siguientes métodos de conversión:
  • Paralelo: Con este enfoque, los usuarios siguen operando el sistema anterior pero a su vez comienzan a utilizar el sistema nuevo. Es el enfoque más seguro, duplica los costos de operación
  • Directo: El sistema nuevo es instalado reemplazando el anterior. Genera beneficios inmediatos de los nuevos métodos y controles, no hay otro sistema al cual recurrir si surge alguna dificultad
  • Piloto: Se implementa el nuevo sistema en un área de la empresa y de esa forma hacer cambios y mejorarlo para la implementación completa, proporciona experiencia y prueba antes de la implantación
  • Por etapa: Se implanta de manera gradual a los usuarios, permite una capacitación más espaciada, un largo periodo de instalación

El mantenimiento del sistema es esencial para que el sistema mantenga su funcionalidad estable y se adapte a los cambios que vayan surgiendo en la empresa o el entorno. El mantenimiento puede dividirse en 4 diferentes tipos:
  • Adaptable: el sistema es modificado para hacer frente a cambios que surgen en la empresa o el entorno
  • Perfectivo: Se agregan nuevos requerimientos referentes a mejoras funcionales para el software
  • Correctivo: Se diagnostican y corrigen errores, posiblemente encontrados por el usuario
  • Preventivo: Se aumentan o mejoran características del software para evitar problemas a futuro

La capacitación de un sistema es de suma importancia a la hora de implementarlo. Es necesario que la capacitación sea completa y totalmente aceptada por los usuarios. Existen diferentes métodos para lograr una capacitación, entre los cuales se encuentran:
  • Mediante un manual de usuario: el manual es desarrollado por la empresa que proporciona el software, y contiene toda la información necesaria para poder utilizar de forma adecuada el sistema
  • De forma directa: los usuarios utilizan el equipo en presencia de los instructores
  • Por soporte web: Se brinda al usuario videos informativos en los cuales se explica de forma detallada el uso del sistema
